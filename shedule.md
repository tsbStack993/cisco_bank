

## Project Schedule 


| Week | Phase                    | Activities                                         | Deliverables              |
| ---: | ------------------------ | -------------------------------------------------- | ------------------------- |
|    1 | Requirement Analysis     | User roles, VLAN needs, security requirements      | SRS document              |
|    2 | Literature Review        | Study bank network designs, VLANs, ACLs, firewalls | Literature review chapter |
|    3 | Logical Design           | VLAN plan, IP addressing, flow logic               | Logical network diagram   |
|    4 | Physical Design          | Switch/router selection, topology mapping          | Physical topology diagram |
|    5 | Flow & Use-Case Design   | Network flow chart, use-case                       | Flow & use-case diagrams  |
|    6 | Implementation – VLANs   | VLAN creation, trunking, access ports              | VLAN configuration        |
|    7 | Implementation – Routing | Inter-VLAN routing on Core L3 switch               | Routing configuration     |
|    8 | Security Configuration   | ACLs, firewall rules, NAT                          | Security configuration    |
|    9 | Testing & Validation     | Connectivity tests, ACL verification               | Test cases & results      |
|   10 | Documentation            | Report writing, screenshots, configs               | Final report draft        |
|   11 | Review & Submission      | Corrections, viva prep, final submission           | Final project submission  |
